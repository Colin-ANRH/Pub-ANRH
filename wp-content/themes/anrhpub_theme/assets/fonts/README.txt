Polices charte graphique ANRH
=============================

Luciole (CC-BY) — téléchargement officiel :
https://luciole-vision.com/fonts/Luciole_webfonts.zip

Copier dans ce dossier (assets/fonts/) :
- Luciole-Regular.woff2
- Luciole-Bold.woff2

Neulis Neue Black Italic — police commerciale (Monotype).
Copier le fichier web fourni par la charte :
- NeulisNeue-BlackItalic.woff2

Sans ces fichiers, le site utilise les polices locales installées
sur le poste, puis Verdana (repli charte).
